# PKRC 
=====================================

What is PKRC?
----------------

PKRC is BEP-20 Token with a total finite suppl of 100 Billion PKRC, 
these tokens can be used on the Binance Smart Chain for value transfers or
could be traded on DEXes like PancakeSwap.
